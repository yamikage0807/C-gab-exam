# C-GAB Practice Kit

本プロジェクトは、適性検査「C-GAB」の本格的な模擬試験・練習を行えるWebアプリケーション（シングルページアプリケーション）です。
HTML、CSS、JavaScript（バニラJS）のみで構成されているため、ビルド不要でポータブルに動作します。

## 主な機能
- **トグルボタン選択:** A〜Eの選択肢をクリックまたはキー入力で直感的に切り替え。
- **後で解く（フラグ）機能:** 迷った問題にフラグを立てて、ナビゲーター上で視覚的に把握。
- **試験タイマー＆カスタマイズ:** カテゴリごとに制限時間、問題数を自由に調整可能。
- **自動結果遷移:** タイマー切れの際に自動で解答を締め切るかどうかを設定可能。
- **結果詳細レビュー:** 全問題の正否、解答スピード、カテゴリ別正解率を美しいアニメーションチャートで分析。詳細なステップバイステップ解説付き。
- **履歴保存:** 過去10回分の解答結果をブラウザ（`localStorage`）に自動保存。

## 使い方（ローカル実行）

本アプリは静的Webアプリですので、以下のいずれかの方法で簡単に動かせます。

### 方法1. ダブルクリックで開く
`index.html` ファイルをブラウザ（Chrome, Edge, Safariなど）にドラッグ＆ドロップ、またはダブルクリックして開いてください。

### 方法2. ローカルサーバーを起動する
開発者向けツール等を利用してローカルサーバーを立ち上げると、より安定して動作します。
```bash
# Node.jsがインストールされている場合
npx serve .
```

## キーボード操作（ショートカット一覧）

試験画面では、実際のWeb試験の操作感に近づけるため、キーボード操作に対応しています。

| キー | アクション |
| :--- | :--- |
| `1` 〜 `5` / `A` 〜 `E` | 各選択肢を選択 |
| `←` (左矢印キー) | 前の問題に移動 |
| `→` (右矢印キー) | 次の問題に移動 |
| `F` キー | 「後で解く（フラグ）」の切り替え |

## GitHub Pagesへのデプロイ方法

本リポジトリの全ファイルをそのままGitHubにアップロードするだけで、GitHub Pagesとして即座にWeb公開できます。

1. GitHubに新しいリポジトリを作成します。
2. 本フォルダ内のファイル（`index.html`, `style.css`, `app.js`, `questions.js`, `README.md`）をプッシュします。
3. リポジトリの **Settings** > **Pages** に移動します。
4. **Build and deployment** の Source にて `Deploy from a branch` を選択し、`main` または `master` ブランチの `/ (root)` を選択して **Save** します。
5. 数分後、提供されたURL（`https://<ユーザー名>.github.io/<リポジトリ名>/`）で公開されます。
